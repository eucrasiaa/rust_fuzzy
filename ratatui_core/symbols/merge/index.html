ratatui_core::symbols

Module merge 

Source
Expand description

This module provides strategies for merging symbols in a layout.

It defines the MergeStrategy enum, which allows for different behaviors when combining symbols, such as replacing the previous symbol, merging them if an exact match exists, or using a fuzzy match to find the closest representation.

The merging strategies are useful for collapsing borders in layouts, where multiple symbols may need to be combined to create a single, coherent border representation.
