Expand description

The type of border of a Block.

See the borders method of Block to configure its borders.

Variants§

§

Plain

A plain, simple border.

This is the default

§Example

│       │
└───────┘

A plain border with rounded corners.

§Example

╭───────╮
│       │
╰───────╯
§

Double

A doubled border.

Note this uses one character that draws two lines.

§Example

║       ║
╚═══════╝
