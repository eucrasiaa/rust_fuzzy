2# Entities Module
3
4the `entities` module does some pre-defined implementations of structs
5with the FuzzyCanidate traits. custom ones can be done via the trait
6pre-defined:
7 - **`DesktopEntity`**: parses/handles Linux `.desktop` files for application launching (theoredically).
8    - parse_desktop_inis()
9 - **`AnimalEnt`**: a simple entity model used for demonstration and scale testing, but left in for fun (yay!)
10 - **`GenericStringStruct`**: A versatile standard struct for basic line strings, with a helper to pass it any file and reading by lines
11
12 - **`CandidateGenerator`**: nice way to convert a Vec<String> into a basic GenericStringStruct type.
13 - **`animal_demo`**: for animals 
14 - **`new_generic_from_file`**: lets you pass a file. see the main demo
15 - **`parse_desktop_inis`**: reads ini files. it kinda works.
16
17*/
